# 🍄 Mushroom Mart

Mushroom Mart is a full-stack e-commerce web application built with Django for selling fresh and medicinal mushrooms.  
It supports multi-role access with separate Admin, Seller, and User dashboards, providing a clean and scalable platform structure.

---

## 🚀 Features

### 👤 User Features
- User registration, login, and logout
- Profile management with image upload
- Address management (add, edit, delete)
- Browse products and view details
- Add to cart and Buy Now purchase
- Order confirmation and order history
- Submit product reviews

### 🛒 Seller Features
- Dedicated seller dashboard
- Seller profile and shop management
- Add, edit, and delete shops
- Product management (add, update, delete)
- View orders related to seller products

### 🛠️ Admin Features
- Centralized admin dashboard
- View and manage all products
- View and manage all orders
- View seller profiles
- Delete or manage sellers
- Platform-wide control and monitoring

---

## 🧱 Tech Stack

- **Backend:** Django 4.x
- **Frontend:** HTML, Bootstrap, JS 
- **Database:** MySQL (configurable)
- **Authentication:** Django Auth System
- **ORM:** Django ORM
- **Media Handling:** Django Media Files
- **Version Control:** Git & GitHub
